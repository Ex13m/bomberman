# 💣 Bomberman — клон на двоих

Браузерная аркада на сетке в духе классического Bomberman. Hot-seat-мультиплеер:
**два игрока на одной клавиатуре и одном экране** ставят бомбы, рушат стены и
стараются взорвать соперника.

## Играть

Нужен локальный статический сервер (ES-модули по `file://` браузер блокирует):

```bash
# любой статик-сервер из корня репозитория, например:
npx serve .
# или
python -m http.server 8000
```

Затем открыть `http://localhost:8000`. В проде раздаётся как статика (Netlify).

## Управление

| Действие | Игрок 1 | Игрок 2 |
|----------|---------|---------|
| Движение | `W A S D` | стрелки |
| Бомба    | `Space`  | `Enter` |
| Рестарт  | `R` | — |

## Геймплей

- Сетка со стенами и разрушаемыми блоками; спавны в противоположных углах.
- Бомба взрывается крестом, гаснет о несокрушимые стены, рушит первый блок на луче.
- Цепная детонация соседних бомб; гибель игрока в пламени.
- Бонусы из блоков: 💣 +бомба, 🔥 +радиус, ⚡ +скорость.
- Побеждает последний выживший; обоюдная гибель — ничья.

## Как это сделано

- **Движок:** Vanilla JS + HTML5 Canvas, ES-модули, без сборки и зависимостей.
- **Структура:** `src/` — `config`, `game`, `player`, `bomb`, `input`, `render`, `audio`, `main`.
- **Звук:** SFX сгенерированы в **Higgsfield** (модель Mirelo Text-to-Audio), лежат в `assets/sfx/`.
- **Старт проекта:** каркас и структура подняты скилом **`/born`** в Claude Code; история
  «рождения» и инкрементов видна по `git log`.

Спека и план инкрементов — в [`kb/kb-spec.md`](kb/kb-spec.md).
